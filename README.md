# JobPortal.kz - Жұмыс іздеу платформасы

Бұл студенттік жоба - жұмыс іздеушілер мен жұмыс берушілерді байланыстыратын заманауи платформа.

## 🚀 Технологиялық стек
- **Frontend**: React.js, React Router, Axios
- **Backend**: Node.js, Express.js
- **Деректер базасы**: PostgreSQL, Sequelize ORM
- **Тестілеу**: Mocha, Chai, Supertest
- **Дизайн**: Custom CSS (Dark/Light mode)

## 📋 Мүмкіндіктер
- Пайдаланушыларды тіркеу және авторизация (JWT)
- Вакансияларды іздеу және сүзу
- Вакансияларды жариялау және басқару (Админ)
- Пайдаланушылар арасындағы Чат
- Хабарландырулар жүйесі
- Түнгі/Күндізгі режим

## 🛠️ Орнату және іске қосу

### 1. Репозиторийді көшіру
```bash
git clone https://github.com/Zhannu1/webka.git
cd webka
```

### 2. Backend орнату
```bash
cd backend
npm install
# .env файлын жасап, DATABASE_URL қосыңыз
npm start
```

### 3. Frontend орнату
```bash
cd ../frontend
npm install
npm start
```

## 🧪 Тестілеу
Backend тесттерін іске қосу үшін:
```bash
cd backend
npm test
```
